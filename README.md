# Máquina De Café 
Implementación de Test Driven Development (TDD)

## Uso
Dentro del directorio del proyecto, abrir la terminal y poner lo siguiente:
```bash
$ npm install
```
 Ejecutar las pruebas
 ```bash
 $ npm test
```
 Ejecutar aplicación
 ```bash
 $ npm start
```
## Tecnologías utilizadas
- NodeJS
- Typescrypt
- Jest
- Inquirer

## Modelos/Metodología
- POO
- TDD
